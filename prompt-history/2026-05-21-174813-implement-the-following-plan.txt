│ 1. Season enum (src/model/season.js)                                         │
│                                                                              │
│ export const Season = Object.freeze({                                        │
│   WINTER: 'WINTER', SPRING: 'SPRING', SUMMER: 'SUMMER', AUTUMN: 'AUTUMN',    │

│ */ }                                                                         │

│                                                                              │
│ 2. WeatherCondition (src/model/weather-condition.js)                         │
│                                                                              │
│ Value object with 5 fields: temperature_C, irradianceFactor (0-1),           │
│ cloudCover (0-1), season, timestamp.                                         │
│                                                                              │
│ 3. DeterministicWeatherModel (src/model/weather-model.js)                    │
│                                                                              │
│ Constructor takes config object with all parameters from YAML weather:       │
│ section + stepSizeMinutes.                                                   │
│                                                                              │
│ Methods:                                                                     │
│ - getWeather(dateTime, rng) → WeatherCondition                               │
│ - _computeTemperature(dateTime, rng) → double — sinusoidal annual + diurnal  │
│ + noise (FR-W3.2)                                                            │
│ - _computeSolarElevation(dateTime) → double — declination + hour angle +     │
│ latitude (FR-W4)                                                             │
│ - _computeIrradiance(dateTime, cloudCover) → double — clear-sky × cloud      │
│ attenuation (FR-W4.2)                                                        │
│ - _computeCloudCover(dateTime, rng) → double — random walk with seasonal     │
│ bias (FR-W5)                                                                 │
│ - _deriveSeason(dateTime) → Season                                           │
│                                                                              │
│ Key formulas (from FR-W3/W4/W5):                                             │
│ - T = annualMean + annualAmp*sin(2π*(doy-80)/365) +                          │
│ (diurnalAmp/2)*sin(2π*(hour-8)/24) + noise                                   │
│ - Solar elevation via declination + hour angle + latitude                    │
│ - irradianceFactor = max(0, sinElevation) * (1 - cloudCover)                 │
│ - Cloud cover: random walk bounded [0,1] with winter/summer bias             │
│                                                                              │
│ Stateful: stores lastCloudCover for smooth random walk transitions.          │
│                                                                              │
│ 4. Asset.step() signature change                                             │
│                                                                              │
│ Breaking change: step(deltaTimeHours, rng, context?) → step(deltaTimeHours,  │
│ weather, rng)                                                                │
│                                                                              │
│ - weather is always a WeatherCondition (replaces ad-hoc context object)      │
│ - rng is the PRNG (kept for assets that need randomness: EV chargers, base   │
│ consumption random walk)                                                     │
│                                                                              │
│ 5. PV model rewrite                                                          │
│                                                                              │
│ Old: random cloud factor + crude day/night + seasonal sin curve              │
│ New: this.currentPower_kW = -(this.peakPower_kW * weather.irradianceFactor)  │
│ — single line driven by weather                                              │
│                                                                              │
│ 6. Heat pump model rewrite                                                   │
│                                                                              │
│ Old: random activity scaled by month-based season factor                     │
│ New: physics-based heating demand driven by outdoor temperature:             │
│ heatingDemand = max(0, indoorTarget - weather.temperature_C)                 │
│ power = min(heatingDemand * thermalLossCoefficient / cop, maxPower_kW)       │
│ New constructor params: maxPower_kW, indoorTargetTemp,                       │
│ thermalLossCoefficient, cop.                                                 │
│                                                                              │
│ 7. Base consumption update                                                   │
│                                                                              │
│ Adds optional time-of-day/season modulation (FR-W8):                         │
│ - Evening bump (18-22h): up to +20%                                          │
│ - Night dip (0-6h): down to -20%                                             │
│ - Configurable via enableBaseLoadModulation flag                             │
│                                                                              │
│ 8. Engine tick sequence (updated)                                            │
│                                                                              │
│ 1. Advance clock                                                             │
│ 2. weather = weatherModel.getWeather(clock.currentTime, rng)                 │
│ 3. neighbourhood.step(deltaTimeHours, weather, rng)                          │
│ 4. Record history                                                            │
│ 5. Emit state (now includes weather)                                         │
│                                                                              │
│ Old context = { month, hour } is removed — weather object carries all needed │
│  context.                                                                    │
│                                                                              │
